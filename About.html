About NYC Wifi Finder

This is a web application, where users can type in the name of the location or address in the search bar and find free wifi zones near inputted location or address.

This web application is developed using Javscript, JQuery, HTML, and CSS.


How to Use

  1. Type in the name of location or address in the search bar. Ex. Empire State Building / 42nd Street New York, NY
  2. The marker on the map will be relocated to the inputted location or address.
  3. Free wifi zones will be marked as new wifi icons.
  4. Click on wifi icons to see detailed information about selected wifi zones; name, location, and location coordinates.
  5. Use "Hide Markers" and "Show Markers" to toggle markers.
